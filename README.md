# Student Performance Prediction

# Problem Statement
This project predicts whether a student will pass or fail based on
academic and social factors using machine learning.

# Objective
- Analyze student data
- Apply preprocessing
- Train ML models
- Evaluate performance

# Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

# Models Used
- Logistic Regression
- Decision Tree
- Random Forest

# Outcome
The model helps understand key factors that affect student performance.
